🧠 MemoMind – AI Notes Assistant

MemoMind is an AI-powered note-taking web application built with Django. It helps users organize, manage, and search their notes intelligently using semantic search powered by sentence embeddings.

✨ Features
🔐 User Authentication (Register, Login, Logout)
📝 Create, Edit, View and Delete Notes
🏷️ Categorize Notes
🤖 AI Category Suggestion
🔎 AI Semantic Search using Sentence Transformers
📊 Dashboard with statistics
🌙 Dark Mode
⚙️ User Settings
📱 Responsive Design
🧠 AI Features
AI Semantic Search

Instead of matching exact keywords, MemoMind converts notes and user queries into embeddings and retrieves the most semantically similar notes using cosine similarity.

AI Category Suggestion

Automatically predicts the most suitable category for a note based on its content.
